We analyzed the implementation‘s effects on the existing software architecture, changes, risks, and limitations, and use cases, and also the lower and higher structures and risks of the system.


UIUX Design Project About a Time Management Web App

A course group project from UI/UX Design W21

The PlantTime project is a Web Application which tends to enforce people's habit of managing time with a healthy work-life-balance lifestyle We studied related articles, competitor applications and user characterizations and user scenarios for the designing process.


Bino-Books E-Commerce Website Feature Development

A Group Project for a Startup company called BinoBooks, Information Technology Project F21

BinoBooks aims is an E-Commerce website to creating personalized e-storybooks. The stories will allows parents to explain complex topics such as COVID-19, racism and more, and the Character customization feature will enable children to engage in more meaningful ways by seeing a representation of themselves and their identity in the stories.
